# Browser Actogram

A simple, privacy-respecting browser extension that visualizes your online activity patterns to help track sleep/wake cycles.

![Browser Actogram Icon](icons/icon-128.png)

## Installation

### Chrome
1. Download/Clone this repository.
2. Go to `chrome://extensions/` and enable **Developer mode** (top right).
3. Click **Load unpacked**.
4. Select the `browser-actogram` folder.

## How to Use
- **View Chart**: Click the extension icon to see your activity actogram.
- **Settings**: Adjust time epochs (default: 15 mins) or export your data.
